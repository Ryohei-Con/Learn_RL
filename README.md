## 概要
このプロジェクトでは、強化学習について広く理解を深めるためにアーキテクチャ、報酬設計を変更しながら結果を観察します。
学習対象はCartPoleです。

## フォルダ構成
main.pyでモデルを学習し、同じディレクトリ内にoutputディレクトリを作り、その中にモデルの重みとrewardの推移を格納しました。
実験はまだ途中です。

## 計画
アーキテクチャとしては、経験再生やターゲットネットワークのないQ学習と方策勾配法を試してみるつもりです。
角度によって報酬関数を変えてみることにも興味があります。

## 番外編
move_cartディレクトリにて、報酬系をうまく設計してカートを左から右へ動かしてみる
